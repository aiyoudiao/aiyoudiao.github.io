# 消元法

消元法来解线性方程组，通过不停的消减未知数，最终得到所有未知数的值。

常用的方式有三种基本操作：

  1. 一个方程的左右两边同时乘以一个常数。
  2. 用一个方程加或减另一个方程组。
  3. 交换两个方程的位置

# 前言

React 就干两件事，打造用户界面，响应各种事件。16.0版本提出fiber，16.8版本提出hooks。
