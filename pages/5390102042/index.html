一个移动端的、微信端的 md 编辑器，同时可以使用你的仓库来做图床：md editor (opens new window)

一个虎年的头像生成器：hu generator (opens new window)

2019 年做的一个基于 mermaid 和 graphviz 的云闪付风格的 md 绘图工具：my live editor (opens new window)

2019 年做的一个 md 编辑器，作者不知道因为什么缘故不维护了，于是我 fork 了他的源码，改了里面的 bug 添加了 graphviz 的思维导图绘制功能。Markdown Plus (opens new window)

2020 年做的公司里传统套模板引擎后台项目的改造方案，这个 github 上有 webpack4 的项目配置方案源码。MVC 版本的 aiyou-admin 方案 (opens new window)

2020 年发现的一个好玩的解压的洛天依音乐开源项目。Music (opens new window)

一个正则表达式可视化的网站，可以帮助开发者更快的了解正则。Regulex (opens new window)

一个可以让你看到的浏览器端不同电脑升级系统的网站。system update game (opens new window)

一个好看的正则大全网站。🦕 正则大全 (opens new window)

一个图片背景一键清除的网站。 图片背景消除 (opens new window)

一个前端代码转 AST 的网站。 AST Explorer (opens new window)

一个有很平面图片的网站。 unDraw (opens new window)

一个制作 md 文档 中小徽章的网站。 shields.io (opens new window)

一个可以看到 数据结构和算法 动态可视化的网站。 数据结构和算法动态可视化 (Chinese) (opens new window)

大圣老师的前端编程自学网。 大圣前端编程自学网 (opens new window)

一个可生成 vscdoe 中的代码判断的半交互式网站：片段生成器 (opens new window)

一个可以生成高亮的代码图片的网站：carbon (opens new window)

一个很不错的工具集网站，常见的前端工具里面都有：在线工具 (opens new window)

一个很不错的图片压缩的工具网站，除了其它图片格式还支持压缩 gif：图片压缩好网站 (opens new window) 图片压缩工具相关的文章 (opens new window)
