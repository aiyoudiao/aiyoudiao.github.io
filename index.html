# JS 性能优化

# 前言

内存管理：申请 -> 使用 -> 释放
垃圾回收：不可达对象（没有使用到且不可访问到的对象）就是垃圾对象，会被JS垃圾回收引擎自动回收。
GC算法：不可达对象的内存空间会被GC算法回收。大概是这四种算法 引用计数、标记清除、标记整理、分代回收。
性能优化：细节上的留心，使程序更自信。
